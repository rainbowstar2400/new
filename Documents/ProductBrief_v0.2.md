# Product Brief v0.2

- 作成日: 2026-02-07
- プロダクト名（仮）: 自分専用秘書PWA
- 方針: オンラインMVP

## 1. 目的

- 会話入力でタスク・メモ・リマインドを一気通貫で扱う
- 確認が必要な場面は選択式 (`設定する/設定しない/後で設定する`, `○/✕`) で確定する
- 忘却を減らすため、Web Pushで通知する

## 2. MVPの主要前提（v0.2）

- フロント: Vite + React + TypeScript
- API: Node.js (Fastify)
- DB: PostgreSQL
- 要約スロット (`〇〇`) は GPT-API 利用
- ユーザー識別は匿名デバイスID
- 保存対象はタスク/メモ/通知関連

## 3. 価値仮説

- 入力負荷を減らし、登録漏れを防ぐ
- 期日設定・時刻補完・オフセット調整を会話で完了できる
- `!期日設定` 表示で「後で設定する」タスクを見落としにくくする

## 4. 非目的（MVP）

- メール認証
- 複数アカウント管理
- 高度な推薦

## 5. 変更点（v0.1 -> v0.2）

- ローカル限定方針を改め、オンラインMVPへ移行
- 外部送信を全面禁止から「最小送信・用途明示」へ改訂
- Web Push配信基盤をMVPに含める

## 6. 送信データ最小化方針

- GPTへ送るのはユーザー入力と確定済み構造化情報のみ
- 未確定情報は送らない
- 監査ログは最小限（入力・応答・時刻）で保持
